# [172. 阶乘后的零](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)

# 题目描述：

给定一个整数 `n` ，返回 `n!` 结果中尾随零的数量。

提示 `n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`



**示例 1:**

```
输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0
```

**示例 2:**

```
输入：n = 5
输出：1
解释：5! = 120 ，有一个尾随 0
```

## JavaScript：

找阶乘后末尾`0`的数量 <==> 找`5`

```JavaScript
const trailingZeroes = n => {
  let count = 0;
  while (n >= 5) {
    count += Math.floor(n / 5);
    n /= 5;
  }
  return count;
};
```
