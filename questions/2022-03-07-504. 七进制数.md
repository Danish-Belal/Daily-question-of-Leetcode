# [504. 七进制数](https://leetcode-cn.com/problems/base-7/)

# 题目描述：

给定一个整数 `num`，将其转化为 **7 进制**，并以字符串形式输出。

 

**示例 1:**

```
输入: num = 100
输出: "202"
```

**示例 2:**

```
输入: num = -7
输出: "-10"
```

**提示：**

- `-107 <= num <= 107`

## JavaScript：

简单模拟。

```JavaScript
const convertToBase7 = num => {
    if (num === 0) return '0';
    // 标识正负号
    let isMinus = false;
    if (num < 0) {
        isMinus = true;
        // 如果是负数，取正数
        num = -num;
    }
    // 存放结果的数组
    const res = [];
    // 余数
    let remain = 0;
    while (num > 0) {
        remain = num % 7;
        num = Math.floor(num / 7);
        // 余数从队头放入数组
        res.unshift(remain);
    }
    // 判断正负号
    return isMinus ? '-' + res.join('') : res.join('');
};
```
